Jquery Forever

Small, industrious, user centric interactions created with Jquery.

Protip: Most interactions include a parent container, which is used for scoping, or in other words - a reference element to match against. This keeps the possiblity of interference from multiple interactions within the same document to a minimum.

Protip: Learn why and how to use document ready states. All examples in this document use the (function($) { })(jQuery); function syntax to execute after the DOM is ready.

Protip: No images are being used for icons in any interaction. Icons are created using CSS.

Simple Toggle

Protip: Do not directly use margin or padding on the targeted element that gets animated, it sometimes causes unsightly animations. Instead, use a nested element if any content positioning is needed.

Example

Target

Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s, when an unknown printer took a galley of type and scrambled it to make a type specimen book.

It has survived not only five centuries, but also the leap into electronic typesetting, remaining essentially unchanged. It was popularised in the 1960s with the release of Letraset sheets containing Lorem Ipsum passages, and more recently with desktop publishing software like Aldus PageMaker including versions of Lorem Ipsum.

Structure

<div class="simple-toggle-container">
    <button type="button" class="simple-toggle-trigger" aria-expanded="false">
        Simple Toggle Trigger
    </button>
    <div class="simple-toggle-target">
        Content goes here ...
    </div>
</div>

CSS

button.simple-toggle-trigger {
    width: 100%;
    margin: 0 0 -1px 0;
    padding: 10px 15px;
    background: white;
    border: 1px solid #dcdcdc;
    text-align: left;
    position: relative;
    cursor: pointer;
}

button.simple-toggle-trigger::before,
button.simple-toggle-trigger::after {
    content: '';
    height: 2px;
    width: 12px;
    background: #999999;
    position: absolute;
    top: 22px;
}

button.simple-toggle-trigger::before {
    right: 15px;
    transform: rotate(-45deg);
}

button.simple-toggle-trigger::after {
    right: 22px;
    transform: rotate(45deg);
}

button.simple-toggle-trigger.open::before {
    transform: rotate(-135deg);
}

button.simple-toggle-trigger.open::after {
    transform: rotate(135deg);
}

.simple-toggle-target {
    display: none;
}

Jquery

$(".simple-toggle-trigger").on('click', (e) => {
    e.preventDefault();

    let target = $(e.target).parent().find(".simple-toggle-target");
    let isOpen = $(e.target).hasClass('open');

    $(e.target).toggleClass('open', !isOpen).attr('aria-expanded', !isOpen);
    $(target).slideToggle('fast');
});

$(".simple-toggle-trigger").on('keydown', (e) => {
    if (e.key === 'Enter' || e.key === ' ') {
        e.preventDefault();
        $(e.target).trigger('click');
    }
});

Simple Toggle & Animate To Top

Protip: Do not directly use margin or padding on the targeted element that gets animated, it sometimes causes unsightly animations. Instead, use a nested element if any content positioning is needed.

Protip: The HTML element has an id of 'top'. This serves as the anchor for which the targeted element will animate to.

Example

Target

Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s, when an unknown printer took a galley of type and scrambled it to make a type specimen book.

Structure

<html lang="en" id="top">

    <!-- Code -->

    <div class="simple-toggle-to-top-container">
        <button type="button" class="simple-toggle-to-top-trigger" aria-expanded="false">
            Simple Toggle Top Top Trigger
        </button>
        <div class="simple-toggle-to-top-target">
            Content goes here ...
        </div>
    </div>

    <!-- Code -->

</html>

CSS

button.simple-toggle-to-top-trigger {
    width: 100%;
    margin: 0 0 -1px 0;
    padding: 10px 15px;
    background: white;
    border: 1px solid #dcdcdc;
    text-align: left;
    position: relative;
    cursor: pointer;
}

button.simple-toggle-to-top-trigger::before,
button.simple-toggle-to-top-trigger::after {
    content: '';
    height: 2px;
    width: 12px;
    background: #999999;
    position: absolute;
    top: 22px;
}

button.simple-toggle-to-top-rigger::before {
    right: 15px;
    transform: rotate(-45deg);
}

button.simple-toggle-to-top-trigger::after {
    right: 22px;
    transform: rotate(45deg);
}

button.simple-toggle-to-top-trigger.open::before {
    transform: rotate(-135deg);
}

button.simple-toggle-to-top-trigger.open::after {
    transform: rotate(135deg);
}

.simple-toggle-to-top-target {
    display: none;
}

Jquery

$(".simple-toggle-to-top-trigger").on('click', (e) => {
    e.preventDefault();

    var target = $(e.target).parent().find(".simple-toggle-to-top-target");

    if ($(e.target).hasClass('open')) {
        $(e.target).removeClass('open').attr('aria-expanded', 'false');
        target.slideUp('fast');
    } else {
        $(e.target).addClass('open').attr('aria-expanded', 'true');
        target.slideDown('fast');

        $("body, html").animate({
            scrollTop: $(e.target).offset().top - 105
        }, 200);
    }
});

$(".simple-toggle-to-top-trigger").on('keydown', (e) => {
    if (e.key === 'Enter' || e.key === ' ') {
        e.preventDefault();
        $(e.target).trigger('click');
    }
});

Simple Toggle & Click To Close

Click anywhere, except the current target, to close the toggle.

Protip: Do not directly use margin or padding on the targeted element that gets animated, it sometimes causes unsightly animations. Instead, use a nested element if any content positioning is needed.

Example

Target

Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s, when an unknown printer took a galley of type and scrambled it to make a type specimen book.

It has survived not only five centuries, but also the leap into electronic typesetting, remaining essentially unchanged. It was popularised in the 1960s with the release of Letraset sheets containing Lorem Ipsum passages, and more recently with desktop publishing software like Aldus PageMaker including versions of Lorem Ipsum.

Structure

<div class="simple-toggle-click-to-close-container">
    <button type="button" class="simple-toggle-click-to-close-trigger" aria-expanded="false">
        Simple Toggle Trigger
    </button>
    <div class="simple-toggle-click-to-close-target">
        Content goes here ...
    </div>
</div>

CSS

button.simple-toggle-click-to-close-trigger {
    width: 100%;
    margin: 0 0 -1px 0;
    padding: 10px 15px;
    background: white;
    border: 1px solid #dcdcdc;
    text-align: left;
    position: relative;
    cursor: pointer;
}

button.simple-toggle-click-to-close-trigger::before,
button.simple-toggle-click-to-close-trigger::after {
    content: '';
    height: 2px;
    width: 12px;
    background: #999999;
    position: absolute;
    top: 22px;
}

button.simple-toggle-click-to-close-trigger::before {
    right: 15px;
    transform: rotate(-45deg);
}

button.simple-toggle-click-to-close-trigger::after {
    right: 22px;
    transform: rotate(45deg);
}

button.simple-toggle-click-to-close-trigger.open::before {
    transform: rotate(-135deg);
}

button.simple-toggle-click-to-close-trigger.open::after {
    transform: rotate(135deg);
}

.simple-toggle-click-to-close-target {
    display: none;
}

Jquery

$(".simple-toggle-click-to-close-trigger").on('click keypress', (e) => {
    if (e.type === 'keypress' && e.key !== 'Enter' && e.key !== ' ') {
        return;
    }

    e.preventDefault();

    let target = $(e.currentTarget).parent().find(".simple-toggle-click-to-close-target");
    let isOpen = $(e.currentTarget).hasClass('open');

    $(e.currentTarget).toggleClass('open', !isOpen).attr('aria-expanded', !isOpen);
    $(target).slideToggle('fast');
});

$(".simple-toggle-click-to-close-trigger").on('keydown', (e) => {
    if (e.key === 'Enter' || e.key === ' ') {
        e.preventDefault();
        $(e.currentTarget).trigger('click');
    }
});

$(document).on('click', (e) => {
    let trigger = $(".simple-toggle-click-to-close-trigger");
    let target = $(".simple-toggle-click-to-close-target");

    if (
        !$(trigger).is(e.target) &&
        !$(trigger).find('*').is(e.target) &&
        !$(target).is(e.target) &&
        !$(target).find('*').is(e.target)
    ) {
        $(trigger).removeClass('open').attr('aria-expanded', 'false');
        $(target).slideUp('fast');
    }
});
